# CSP451-CheckPoint1-RayChang

## Description
This repository is the submission for the CSP451 Week 1 Checkpoint. It demonstrates the fundamentals of version control, including creating repositories, managing commits, and working with GitHub.

## Installation
1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open `index.html` in any web browser.

## Usage
Once the `index.html` file is open, right-click the page and select **Inspect** to open the Developer Tools. Navigate to the **Console** tab to view the JavaScript output.
